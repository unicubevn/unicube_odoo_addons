Bean Vietnamese Address Module

This is the Vietnamese address module based on Circular No 124/2004/QĐ-TTg issued by Viet Nam Government (for backend sites only).
As the code of state, district and ward are aligned with Government requirements, this module can be used the based module for other modules that need the address information is the same as the one used in Government application like tax report application, VAT invoice issuing applications...

Change logs

  • Version:16.0.0.2

    Add checking whether the Python Redis is installed or not. If Python Redis is installed, you can use redis cache for APIs in Controller Class.

    To Use Redis, please add following config to your Odoo config file:

                        
      
                    
      enable_redis = True
      redis_host = localhost # Redis host
      redis_port = 6379      # Redis port
      redis_dbindex = 2      # Redis database index
      redis_pass =           # Redis password (default: None)
      redis_expired_time = 604800 # in second
